Get started

Installation:

RISE is HTML template so there's no really "installation" required just unzip the main file you've downloaded from themeforest and after unziping you have your all template ready to be used.

Setting Up The Grid:

Before starting to work deeper with this template, you'll need to know how to use template's grid, it's simple and easy to use. RISE is being build using one of the most popular grid, the 960 grid, which means your site container is 960px wide.To make it work without problem you be using columns for everything starting from sliders , portfolio , blog and more.

Changing colors

Easiest way: 1) Open style/color folder – There is 18 css file (1 file 1 color scheme, name of file – color ) 2) Open one of them and copy all to clipboard (ctrl+c) 3) Open css folder and open there style.css 4) In the BOTTOM of style.css past from clipboard (ctrl+v) and safe file Thats all
